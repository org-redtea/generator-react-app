# Installing

Install yeoman

```bash
yarn global add yo
```

Install generator

```bash
yarn global add @redtea/generator-react-app
```

# Usage

Run generator

```bash
yo @redtea/react-app <project-name> --force
```

Go to project

```bash
cd <project-name>
yarn start
```

Print generator options
```bash
yo @redtea/react-app --help
```

# Features
 
 - Based on Create React App TypeScript template
 - Choosing statics server engine:
    - NodeJS (serve)
    - Nginx
 - Preinstalled packages:
    - axios
    - classnames
    - history
    - inversify
    - reflect-metadata
    - inversify-react
    - react-router-dom
    - mobx
    - mobx-react
 - Components examples
